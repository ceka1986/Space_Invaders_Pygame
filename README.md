🚀 Space Invaders - Python & Pygame ProjectA classic 2D arcade shooter developed to demonstrate Object-Oriented Programming (OOP) principles in Python. This project features autonomous enemy AI, collision detection, and dynamic game-state management. 🛠 FeaturesOOP Architecture: Built using modular classes for the Player, Enemies, and Projectiles to ensure clean and maintainable code. Dynamic UI: Implemented a real-time health bar for the player and a countdown system before gameplay begins. Enemy AI: Automated enemy movement patterns with randomized shooting logic. Asset Integration: Custom sound effects for lasers and explosions, along with sprite animations for game entities. 🧪 Tech StackLanguage: Python Library: Pygame (Surface rendering, Sprite groups, Collision detection) Dev Ops: Virtual Environments (venv) and Dependency Management (pip) 📋 How to RunClone the repository:Bashgit clone https://github.com/ceka1986/space-invaders-pygame.git
cd space-invaders-pygame
Set up a Virtual Environment (Linux):Bashpython3 -m venv venv
source venv/bin/activate
Install Dependencies:Bash pip install -r requirements.txt
Run the Game:Bashpython main.py
