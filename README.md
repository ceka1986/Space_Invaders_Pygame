# 🚀 Space Invaders - Python & Pygame Project

A classic 2D arcade shooter developed to demonstrate **Object-Oriented Programming (OOP)** principles in Python. This project features autonomous enemy AI, collision detection, and dynamic game-state management.

## 🛠 Features

- **OOP Architecture**: Built using modular classes for the Player, Enemies, and Projectiles to ensure clean and maintainable code.
- **Dynamic UI**: Implemented a real-time health bar for the player and a countdown system before gameplay begins.
- **Enemy AI**: Automated enemy movement patterns with randomized shooting logic.
- **Asset Integration**: Custom sound effects and sprite animations for game entities.

## 🧪 Tech Stack

- **Language**: Python
- **Library**: Pygame (Surface rendering, Sprite groups, Collision detection)
- **Dev Ops**: Virtual Environments (`venv`) and Dependency Management (`pip`)
